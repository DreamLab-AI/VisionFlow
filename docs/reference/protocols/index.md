## Protocol Overview

VisionFlow supports multiple communication protocols:

1. **Binary WebSocket** - High-performance real-time updates
2. **JSON WebSocket** - Human-readable message format
3. **REST HTTP** - Standard CRUD operations
4. **MCP** - Model Context Protocol for AI integration
