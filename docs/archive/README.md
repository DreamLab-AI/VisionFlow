## Archived Content Guidelines

Content is archived when:
- Completion reports (migration, implementation summaries)
- Sprint logs and retrospectives
- Implementation tracking documents
- Deprecated guides superseded by newer versions
- Temporary working documents

Archived content is preserved for historical reference and audit trails.
