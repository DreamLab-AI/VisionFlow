## What was written

`docs/protocol/voice-addressing.md` — the addressing counterpart to
`identity-spine.md` (which is DID-focused and does not cover addressing). It defines
the multi-agent voice addressing / turn-taking model the task specifies:

- **PTT-to-selected-actor as the shipped primitive (§2).** COM-15 (landed P1) is the
  base case: selection is the address, resolved to the actor's `did:nostr` (COM-14);
  the transcript never re-targets; the utterance dispatches a scoped, signed ACSP
  kind-31402 under the operator's key.
- **Addressed-utterance semantics (§3.1).** A PTT utterance is addressed to exactly
  the selected actor; no selection ⇒ read-only fallback, never a mutation; identity
  and authority derive from the operator's key, never the transcript.
- **Turn-taking when multiple agents could respond (§3.2).** Exactly one actor holds
  the floor (single floor token = the selection); no two recognition paths acting on
  the same audio (fixes the register's "two competing swarm-keyword paths … no floor
  management"); deterministic arbitration by selection, not by which path fires
  first; a bidirectional listening cue so the human sees which agent is addressed.
- **Clarification / repair turn (§3.3).** An ungrounded utterance MUST trigger a
  repair turn (a clarifying question), never silent degradation; the repair preserves
  the address and the floor. **VisionClaw lands this as V3 this wave**; the canon
  states the model, VisionClaw implements it.
- **Floor-control rules (§3.2).** Explicit floor hold/release; barge-in is an
  explicit operator action, never an agent-initiated seizure.

## Falsification exposure

Falsified if a PTT utterance may address more than one agent or be re-addressed from
the transcript, if an unselected utterance may mutate agent state, if two recognition
paths may act on the same audio without a single floor token, if the human is not
shown which actor is addressed, if an ungrounded utterance degrades to silence
instead of a repair turn, or if the page is claimed `integrated` before COM-15 cites
it by name.
