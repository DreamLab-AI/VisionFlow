	- definition:: Text-to-Text Transfer Transformer: a unified framework that converts all NLP tasks into a text-to-text format, using an encoder-decoder architecture with a span-corruption pre-training objective.

## Characteristics

- **Unified Framework**: All tasks framed as text-to-text
- **Encoder-Decoder Architecture**: Full transformer with both components
- **Span Corruption**: Pre-training objective masking spans of text
- **Flexible Task Formulation**: Single model for all NLP tasks

## Academic Foundations

**Primary Source**: Raffel et al., "Exploring the Limits of Transfer Learning with a Unified Text-to-Text Transformer", arXiv:1910.10683 (2019)

**C4 Dataset**: Introduced Colossal Clean Crawled Corpus for pre-training.

## Technical Context

T5 explores transfer learning techniques systematically, comparing various pre-training objectives, architectures, and datasets. The text-to-text framework allows the same model, loss function, and hyperparameters to be used across all tasks.

## Ontological Relationships

- **Broader Term**: Pre-trained Language Model
- **Related Terms**: BERT, GPT, Encoder-Decoder Architecture
- **Variants**: mT5 (multilingual), ByT5 (byte-level)

## Usage Context

"T5's text-to-text framework enables a single model to handle translation, summarisation, classification, and question answering with the same architecture."
