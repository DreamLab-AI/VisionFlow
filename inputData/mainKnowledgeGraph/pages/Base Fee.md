## Base Fee

Base Fee refers to the minimum gas price per unit required for transaction inclusion in a block, introduced by eip-1559. dynamically adjusts based on block gas usage, increasing if blocks exceed target size and decreasing if below target. base fee is burned, not transferred to validators.
