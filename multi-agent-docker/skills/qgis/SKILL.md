# QGIS Geographic Skill

Control QGIS Desktop for GIS operations, project management, layer manipulation, and spatial processing.

## Capabilities

- Create and load QGIS projects
- Add/remove vector and raster layers
- Execute processing algorithms
- Run Python code in QGIS context
- Manage map styling and symbology
- Export maps and data

## Tool Functions

- `load_project`: Load QGIS project file
- `add_vector_layer`: Add vector data layer
- `add_raster_layer`: Add raster data layer
- `run_processing`: Execute QGIS processing algorithm
- `execute_python`: Run Python code in QGIS
- `export_map`: Export current map view

## Prerequisites

QGIS 3.X with socket server plugin active on port 2801.
